# Plans Directory

This directory holds research reports and implementation plans for Lancer features.

## Naming Convention

- **Research reports:** `[feature]-research.md`
- **Implementation plans:** `[feature]-plan.md`

## Current Plans

_No plans yet. The first one will be created during Milestone 2 (Email Capture Upgrade)._

## How Plans Are Created

1. **Feature Researcher** creates `[feature]-research.md`
2. **Feature Planner** creates `[feature]-plan.md`
3. **Senior Reviewer** improves `[feature]-plan.md`
4. **Feature Builder** implements from the plan
5. **Code Reviewer** verifies against the plan

See `docs/workflows/feature-workflow.md` for the full process.
